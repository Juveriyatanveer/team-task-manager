TEAM TASK MANAGER

Project Description:
Team Task Manager is a full-stack web application where users can create projects, assign tasks, and track task progress with role-based access control.

Technologies Used:
- Java
- Spring Boot
- MySQL
- HTML
- CSS
- JavaScript
- Maven

Features:
- User Registration
- User Login
- Project Creation
- Task Creation
- Task Assignment
- Status Tracking
- Dashboard

Database:
MySQL database using XAMPP.

APIs:
1. /users/register
2. /users/login
3. /projects/create
4. /projects/all
5. /tasks/create
6. /tasks/all
7. /tasks/updateStatus
8. /tasks/dashboard

How to Run:
1. Start XAMPP MySQL
2. Open project in Eclipse
3. Run TaskmanagerApplication.java
4. Open browser
5. Go to http://localhost:8088
